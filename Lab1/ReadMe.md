## Описание предметной области

**Предметная область:**  
Система поддержки пользователей в вопросах здоровья и медицинской информации.

### Основные сущности предметной области:

- Пользователь (Пациент)
- Ассистент
- Медицинская история
- Симптом
- Лекарство
- Рекомендация
- Врач
- Клиника

### Основные операции системы:

1. Регистрация пользователя
2. Запись симптомов
3. Предоставление медицинских рекомендаций
4. Напоминание о приеме лекарств
5. Поиск клиники по району
6. Консультация с врачом
7. Просмотр медицинской истории

### Базовый класс Person (person.py)
Атрибуты: id, name, phone (приватные с property)

Методы: геттеры/сеттеры с валидацией

Исключения: EmptyValueError при пустых значениях

### Класс Patient (patient.py)
Наследуется от Person

Атрибуты: age, medical_history

Валидация: возраст > 0 (InvalidDataFormatError)

### Класс Doctor (doctor.py)
Наследуется от Person

Атрибуты: specialization, clinic

Валидация: непустые значения

### Медицинские классы (medical.py)
Symptom: name, severity (1-10), date

Medication: name, dosage, schedule

Recommendation: text, source, date

MedicalHistory: симптомы, лекарства, рекомендации

### Класс MedicalAssistant (medical_assistant.py)
Основные операции:

Регистрация пользователей

Запись симптомов

Предоставление медицинских советов

Добавление напоминаний о лекарствах

Поиск клиник по району

Консультации с врачами

### Класс DataStorage (storage.py)
Загрузка/сохранение данных в JSON

Работа с клиниками, врачами, симптомами
