# Arkanoid на Python (pygame)

## Описание
Проект представляет собой реализацию игры Arkanoid на языке Python с использованием библиотеки `pygame`. Приложение разработано в событийно-ориентированном стиле и соответствует требованиям лабораторной работы.

## Реализованные возможности
- главное меню: **начать игру**, **таблица рекордов**, **справка**, **выход**;
- не менее **5 уровней**;
- уровни и параметры игры вынесены во внешние **JSON-конфиги**;
- фоновая музыка и игровые звуки;
- таблица рекордов с сохранением результата;
- анимации разрушения кирпичей и потери жизни;
- справка с правилами игры;
- бонусы: увеличение платформы и дополнительная жизнь.

```bash
pip install -r requirements.txt
python main.py
```

## Управление
- `←` / `A` — движение платформы влево
- `→` / `D` — движение платформы вправо
- `Enter` — подтверждение выбора
- `Esc` — возврат в меню из разделов справки и рекордов

## Формат уровней
Уровни хранятся в файле `config/levels.json`.
Каждый уровень задаётся матрицей строк, где символы обозначают типы кирпичей:
- `0` — пустая ячейка;
- `1`, `2` — обычные кирпичи;
- `3`, `4`, `5` — прочные кирпичи;
- `9` — неразрушимый кирпич.

## Таблица рекордов
Рекорды сохраняются в `config/records.json`. После завершения игры, если результат выше лучшего текущего результата, игроку предлагается ввести имя и сохранить новый рекорд.
