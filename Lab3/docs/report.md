# Отчёт по лабораторной работе

## Тема
Разработка игрового графического приложения **Arkanoid** на языке Python с использованием библиотеки **pygame**.

## Цель работы
Изучить событийно-ориентированное программирование с использованием библиотеки pygame.

## Постановка задачи
Требовалось разработать игру Arkanoid, в которой:
- используется библиотека pygame;
- приложение строится на основе событийно-ориентированного подхода;
- имеются меню, таблица рекордов, справка и выход;
- уровни и игровые параметры хранятся во внешних конфигурационных файлах;
- реализовано не менее пяти уровней;
- присутствуют звуки, музыка и анимации;
- результаты игрока сохраняются в таблице рекордов.

## Используемые технологии
- Python 3
- pygame
- JSON
- Markdown

## Архитектура программы
Приложение разбито на несколько модулей:
- `main.py` — точка входа в программу;
- `src/game.py` — основная игровая логика;
- `src/entities/` — классы игровых сущностей;
- `src/records.py` — работа с таблицей рекордов;
- `src/config_loader.py` — загрузка и сохранение конфигураций;
- `src/animations.py` — система частиц для анимаций;
- `src/ui.py` — отрисовка текста и интерфейса.

## Событийно-ориентированная модель
Приложение работает в основном цикле `pygame`, в котором постоянно обрабатываются события:
- закрытие окна;
- нажатия клавиш в меню;
- нажатия клавиш во время игры;
- ввод имени при сохранении рекорда.

Таким образом, поведение программы определяется поступающими от пользователя событиями, что соответствует событийно-ориентированному программированию.

## Хранение конфигурации
Все основные настройки игры вынесены во внешние JSON-файлы:
- `game.json` — размеры окна, цвета, скорость, количество жизней;
- `levels.json` — описание уровней и типов кирпичей;
- `records.json` — таблица рекордов.

Такой подход позволяет изменять параметры игры без изменения исходного кода программы.

## Реализация уровней
В программе реализовано 5 уровней. Каждый уровень представляет собой массив строк, где символы определяют расположение и тип кирпичей.

## Реализация анимаций
В игре реализованы анимации:
- разрушение кирпича с помощью частиц;
- эффект потери жизни;
- плавный переход между уровнями.

## Звуковое сопровождение
Используются:
- фоновая музыка;
- звук удара о кирпич;
- звук удара о платформу;
- звук потери жизни;
- звук завершения игры.

## Таблица рекордов
После окончания игры результат сравнивается с лучшим существующим рекордом. Если новый результат выше, пользователь может ввести своё имя и сохранить его в таблице рекордов.

## Вывод
В ходе выполнения лабораторной работы была разработана игра Arkanoid с использованием библиотеки pygame. В проекте применён событийно-ориентированный подход, реализовано меню, уровни, таблица рекордов, звуки, музыка и анимации. Все ключевые параметры вынесены во внешние конфигурационные файлы.
